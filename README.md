## Gaining familiarity with pushing from Terminal in VSCode
1) Use "git clone" to access the repo in VSCode.
2) Use "git add" followed by listing all of the new/updated files or by " ." to stage all of the files.
3) Use "git commit -m" to commit, which will allow to add a message in quotes explaining the changes (a further description can be added by adding an additional -m followed by the description in quotes).
4) Use "git push origin main" to push all the changes from the local computer to the main branch.
