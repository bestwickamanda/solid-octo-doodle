## Gaining familiarity with pushing from Terminal in VSCode
After using "git clone" to access the repo in VSCode, I am able to make changes and push it to the main file.
Instead of using "git add" and listing each file, it is helpful to use "git add ." to stage all of the files.
Then, to commit use "git commit -m", which will allow to add a message in quotes explaining the changes (a further description can be added by adding an additional -m followed by the description in quotes).
Lastly, the changes can be pushed from the local computer to the main using "git push"
